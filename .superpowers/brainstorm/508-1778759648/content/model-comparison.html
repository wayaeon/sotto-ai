Background Compute — How We Keep It Near Zero

The Python STT sidecar loads the model into RAM once at startup, then stays fully idle (0% CPU) until a hotkey fires. Recording captures raw audio in a ring buffer, transcription runs only on the captured segment, then the process returns to idle. On your AMD Ryzen AI 7 350 with 32GB RAM, whisper-large-v3-turbo loads in ~3s once, then transcribes a 10-second segment in ~1.2s — the rest of the time: zero CPU, ~2GB RAM pinned.

LLM Cleanup Auto-Selection

If Ollama is installed and ≥16GB RAM available → Qwen3:7B locally (free, private). If 8–16GB → Qwen3:4B. If <8GB → skip cleanup OR use Claude Haiku API (cloud tier only). No cleanup still gives great output — Whisper punctuation is surprisingly clean on clear audio.
